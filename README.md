# 🌌 Lumina – Image Search Web App

Lumina is a modern and responsive image search web application that allows users to explore high-quality images based on keywords. It features a clean UI, real-time search suggestions, image preview modal, and download functionality.

---

## 🚀 Live Demo

🔗 [https://your-live-link-here](https://lumina-image-search-app.vercel.app/)

---

## ✨ Features

* 🔍 **Search Images** – Find high-quality images instantly
* 💡 **Smart Suggestions** – Real-time search suggestions while typing
* 🖼️ **Image Preview Modal** – Click to view images in larger size
* ⬇️ **Download Images** – Download images directly with one click
* 🔄 **Load More Pagination** – Fetch more images seamlessly
* ⚡ **Fast & Responsive UI** – Optimized for all devices
* 🌙 **Modern Dark Theme UI**

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3**
* **JavaScript (Vanilla JS)**
* **Pixabay API** (for image data)
* **Datamuse API** (for search suggestions)

## ⚙️ Setup & Installation

1. Clone the repository

```bash
git clone https://github.com/MAYANKCH0/LUMINA-IMAGE_SEARCH_APP.git
```

2. Navigate to the project folder

```bash
cd LUMINA-IMAGE_SEARCH_APP
```

3. Open `index.html` in your browser
